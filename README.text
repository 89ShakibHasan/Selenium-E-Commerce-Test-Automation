# 🧪 Selenium E-Commerce Testing Framework  

A comprehensive **Selenium automation testing framework** built using **Java**, **Maven**, and **TestNG** — designed with the **Page Object Model (POM)** pattern for scalability and maintainability.  

---

## 🚀 Project Overview  

This framework automates key user journeys of an **e-commerce website** to ensure core functionality and user experience remain consistent across updates.  

### ✨ Key Features  

- ✅ **Selenium WebDriver with Java** for UI automation  
- 🧩 **Maven** for dependency management and build automation  
- 🧪 **TestNG** for test management, grouping, and parallel execution  
- 🧱 **Page Object Model (POM)** for clean code and reusability  
- 🌐 **Cross-Browser Support** (Chrome, Edge, Firefox, etc.)  
- 🕶️ **Headless Execution** for CI/CD integration  
- 📊 **ExtentReports** with screenshots for failed test cases  

---

## 🧰 Functionalities Tested  

| Category | Test Description |
|-----------|------------------|
| 📝 Registration | Create a new user account |
| 🔐 Login | Authenticate existing users |
| 🚪 Logout | End user session securely |
| 🌍 Language Change | Switch website language |
| 📧 Newsletter | Subscribe using email |
| 🔍 Search | Search for products |
| 💖 Wishlist | Add and remove items |
| 🛒 Cart | Add and remove items |
| 💳 Checkout | Complete purchase flow |
| ❌ Account Deletion | Close user account |
